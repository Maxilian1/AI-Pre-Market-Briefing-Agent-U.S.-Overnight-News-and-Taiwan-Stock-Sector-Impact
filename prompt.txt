    a BASKET:* marker,
    a PROXY:* marker,
    or empty for unmapped rows.

- directional_impact_label should be one of:
    potentially_positive
    potentially_negative
    neutral
    irrelevant
    unmapped

Rules:
- If sentiment_label is positive and relevance_score > 0, directional_impact_label = potentially_positive.
- If sentiment_label is negative and relevance_score > 0, directional_impact_label = potentially_negative.
- If sentiment_label is neutral or mixed, directional_impact_label = neutral.
- If relevance_label is irrelevant or sector is Irrelevant, directional_impact_label = irrelevant and no Taiwan mapping rows should be created by default.
- Do not use words like buy, sell, guaranteed, will rise, or will fall.
- Use cautious language only.

Impact score:
- impact_score = sentiment_score * relevance_score * mapping_confidence
- Clamp impact_score to [-1, 1].
- combined_confidence = signal_confidence * mapping_confidence
- Clamp combined_confidence to [0, 1].
